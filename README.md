# pokebox

A Flutter app that lets you explore Pokémon data through the PokeAPI. Search, view stats, abilities, and types with a clean, responsive UI

## Features
PokeBox is a Flutter-based mobile app that lets users explore the Pokémon universe using the PokeAPI. Search for Pokémon, view their stats, abilities, types, and more with a sleek, user-friendly interface. Built for multiplatform.
